# Signup / Profile UI Design

## Goal

Add a minimal signup page that collects the profile attributes required by AuthFoundation.

## Scope

- `/signup` page
- Fields: email, password, name, birth date
- Link between login and signup
- Submit to `POST /signup`

## Out of Scope

- Terms consent display
- Email verification
- Password reset UX

## Validation

- `npm run typecheck`
